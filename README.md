## Maquetación de estilo WhatsApp — Entrega
Descripción breve

Este proyecto consiste en la maquetación en HTML y CSS de una interfaz visual inspirada en WhatsApp Web.

Se reprodujo la estructura principal de la aplicación, incluyendo:

Sidebar lateral con perfil del usuario, acciones, buscador y lista de chats

Panel principal con encabezado del chat, mensajes diferenciados (emisor y receptor) y barra de escritura

El proyecto es visual, sin lógica ni funcionalidades reales.

Tecnologías utilizadas

HTML5

CSS3

Google Fonts (Roboto)

Git (control de versiones)

Estructura del proyecto
/index.html
/styles/style.css
Cómo visualizar el proyecto

Clonar o descargar el repositorio

Abrir el archivo index.html en cualquier navegador web

No requiere instalación ni dependencias adicionales.

Decisiones de diseño / Consideraciones

Se utilizó CSS Grid para la estructura principal de la aplicación, separando el sidebar del panel de chat

Se empleó Flexbox para la organización interna de componentes como headers, lista de chats, mensajes y footer

Se definieron variables CSS para colores y tipografía, facilitando la visual y el mantenimiento

La tipografía elegida fue Roboto, importada desde Google Fonts

Los mensajes se diferenciaron visualmente según el emisor mediante clases específicas

Se agregó un media query para pantallas menores a 880px, ocultando el sidebar y priorizando el chat principal
